# 代码审查清单（Code Review）

> 交付前用另一个模型逐项过一遍，对话截图存档 → 考核「AI Code Review 报告」(5%)。

## 通用
- [ ] 目录结构清晰，符合 Next.js / Flask 规范
- [ ] 无写死的密钥/地址（走环境变量）
- [ ] `.env`、`node_modules`、`.venv` 已在 .gitignore

## 鉴权 & 安全
- [ ] 密码只存哈希，不明文
- [ ] JWT_SECRET 走环境变量，token 设过期
- [ ] 所有业务接口校验 JWT，未授权→401
- [ ] 数据严格按 user_id 隔离，越权→404
- [ ] 数据库不对公网暴露

## 后端
- [ ] 每个接口有输入校验，非法→400 而非 500
- [ ] 删除/更新不存在资源→404
- [ ] 金额非数字/负/0/超大、日期非法均被拦截
- [ ] DB 会话正确提交/回滚

## AI 服务
- [ ] 无 Key / 超时 / 异常 / 非法 JSON 均降级，不崩
- [ ] parse 失败返回 unknown
- [ ] 上下文长度可控

## 前端
- [ ] 6 路由跳转正常，受保护页未登录跳 /login
- [ ] API loading / error / empty 三态处理
- [ ] 表单前端校验，与后端一致

## 健壮性场景（老师会点开测）
- [ ] 输入空 / 纯空格 / 超长(200+) 不崩
- [ ] 重复提交、快速点击无副作用
- [ ] 接口异常不白屏

## 产出
- 问题清单 → 逐条修复（对应 fix: 提交）→ 审查对话截图存档
