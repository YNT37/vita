# Vita 视频演示大纲（约 6–8 分钟）

> 录屏建议：浏览器全屏 + 清晰分辨率；提前打开 https://vita.sanseven.top/  
> 备用：本地 Docker `http://localhost`（若公网卡顿）  
> 口播可按「旁白」栏读，操作按「画面」栏做。

---

## 0. 开场（30 秒）

| 画面 | 旁白 |
|------|------|
| 仓库 README / 线上首页 | 作品名 **Vita · AI 生活管家**；技术栈 Next.js + Flask + PostgreSQL + Docker；线上地址 `vita.sanseven.top`（HTTPS）。 |
| 点开 docs/资料索引 | 文档含需求、架构、API、Prompt 日志、研发日志，满足工程实践考核材料要求。 |

---

## 1. 注册登录与角色（40 秒）

| 画面 | 旁白 |
|------|------|
| 注册新账号 → 登录 | 多用户 JWT 鉴权，数据按用户隔离。 |
| 首页选角色（管家 / 奴才 / 毒舌闺蜜 / 暖心恋人） | AI 语气随角色变化，后续对话用同一角色。 |

---

## 2. AI 自然语言记账（90 秒）★核心

| 画面 | 旁白 |
|------|------|
| 对管家说：`午饭花了 28 元` | 出现**确认卡**，可改金额/分类再写入，避免 AI 误记。 |
| 再说：`花呗支付 50` 或 `京东白条支付 30` | 识别信用账户付款：记流水 + **负债欠款增加**；若无周期提醒会提示还款日。 |
| 确认账户/提醒卡 | 强调：聊天不直接落库，必须用户点确定。 |

可选一句：`新建信用账户叫合花，欠 200，每月 15 号还` → 统计里出现账户 + 每月提醒。

---

## 3. 记账页与统计跳转（70 秒）

| 画面 | 旁白 |
|------|------|
| 打开「记账」看本月流水 | 展示收入/支出列表与账户标签。 |
| 「统计中心」概览 | 点**本月收入/支出**跳到筛选流水；点**资产/负债**进账户 Tab。 |
| 点某个账户名 | 跳到该账户相关流水，体现「统计 → 明细」闭环。 |

---

## 4. 信用账户与还款日（50 秒）

| 画面 | 旁白 |
|------|------|
| 统计 → 账户 → 编辑/新建负债 | 手动填**还款日**（必填）、账单日（可选）。 |
| 保存后打开「提醒」 | 自动同步「还××」每月提醒，和 App 账单节奏对齐。 |

---

## 5. 提醒与通知（40 秒）

| 画面 | 旁白 |
|------|------|
| 提醒列表：待办 / 账单 | 支持完成、周期提醒。 |
| 「我的」→ 浏览器通知（可选 Server酱） | 到期可弹窗；Server酱为可选微信通道。 |
