├── deploy/Caddyfile         # 反代 + HTTPS 域名

## 4. 不要提交 / 已忽略

- `backend/.env`、`deploy/postgres.env`（密钥）
- `node_modules/`、`.next/`、`__pycache__/`、`*.db`
- 本机 IDE 缓存（`.idea/`、`.vscode/`）

本地清理示例（PowerShell）：

```powershell
Remove-Item -Recurse -Force frontend\.next, backend\__pycache__ -ErrorAction SilentlyContinue
Get-ChildItem -Recurse -Directory -Filter __pycache__ | Remove-Item -Recurse -Force
```
