返回 [部署索引](./README.md)。

## 浏览器打不开 / 连不上 API

- 云服务器：安全组是否放行 **80**？应访问 `http://IP/`，不是 `:3000`。  
- 本机：Docker Desktop 是否在跑？`docker compose ps` 是否都 Up？  
- `curl -s http://127.0.0.1/api/health`

## 构建失败 / 内存不足

小规格云主机 `npm run build` 易 OOM：加 swap，或换稍大内存套餐。

```bash
sudo fallocate -l 2G /swapfile
sudo chmod 600 /swapfile
sudo mkswap /swapfile
sudo swapon /swapfile
```

## 改完 `.env` 不生效

```bash
docker compose up -d --build
# 或
docker compose restart backend
```

Postgres 版把命令里的 compose 文件换成 `docker-compose.postgres.yml` 并带上 `--env-file deploy/postgres.env`。

## 换数据库后登录不上

空库需重新注册。

## SSH 几分钟断一次

```
Host *
  ServerAliveInterval 30
  ServerAliveCountMax 6
```

## 上线检查清单

- [ ] 密钥已改  
- [ ] 80 已放行（云）  
- [ ] health 正常  
- [ ] 能注册登录记一笔
