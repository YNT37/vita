## 4. 遇到的问题与解决
- **`python` 被 Windows 商店别名劫持**（`python -m venv` 报 Python not found）：改用 `E:\Env\Python\Python312\python.exe` 启动后端；依赖通过 `pip` 安装。
- **PowerShell 把 JSON 传给 `curl.exe` 时丢引号**：改用 `Invoke-RestMethod` 自测；前端浏览器测试无此问题。
- **主力 agent 谎报「文件已创建」但实际磁盘为空**：改为由单一 Cursor agent 编写，每步用 `dir`/回读验证落盘。
- **cmd 与 PowerShell 命令差异**：`#` 注释、`Copy-Item` 在 cmd 不可用；Git 提交在 cmd 可正常执行。

## 5. 健壮性 / 已覆盖的异常场景
- 注册：用户名空/超长→400、重复→409、密码 <6 位→400
- 记账：金额非数字/≤0/超上限→400、type 非法→400、日期非法→400
- 提醒：title 空/超长→400、due_at 非法→400、type 非法→400
- 鉴权：无 token / 无效 token→401；越权/不存在→404
- 前端：表单前端校验 + 后端错误透传（`ApiError`）；loading / error / empty 三态
