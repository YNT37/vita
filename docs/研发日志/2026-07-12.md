## 4. 遇到的问题与解决
- **`python` 被 Windows 商店别名劫持**（`python -m venv` 报 Python not found）：改用真实解释器全路径 `E:\Env\Python\Python312\python.exe` 运行，依赖已由 `pip` 装到全局，无需 venv 即可启动。
- **PowerShell 把 JSON 传给 `curl.exe` 时丢引号**，导致 body 解析为空、误报"用户名为空"：确认非代码问题，改用 PowerShell 原生 `Invoke-RestMethod` 自测，happy path 全部通过。

## 5. 健壮性 / 已覆盖的异常场景
- 注册：用户名空/超长→400、重复→409、密码 <6 位→400
- 记账：金额非数字/≤0/超上限→400、type 非法→400、日期非 `YYYY-MM-DD`→400
- 鉴权：无 token / 无效 token / 过期→统一 401；越权/不存在→404
